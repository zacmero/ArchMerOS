# Reference Notes

## Visual Reference

Primary desktop-shell inspiration:

- `ikhsan3adi/desktop-dotfiles`

Reference URL:

- <https://github.com/ikhsan3adi/desktop-dotfiles?tab=readme-ov-file>

Important note:

- the reference repo is based on `sway`, not Hyprland

What is being adopted:

- layered translucent look
- thin framed panel treatment
- compact top bar
- information-dense but restrained modules
- centered launcher styling
- lightweight supporting tools

What is not being copied blindly:

- exact window-manager semantics
- exact package choices
- Catppuccin palette

ArchMerOS will adapt the visual language to its own palette and workflow.

## Terminal Reference

The current `mero_terminal` setup already provides the right baseline:

- minimal WezTerm window
- restrained transparency
- no visible tab bar
- shell-first interaction model
- Yazi already integrated for terminal file work

ArchMerOS should complement this instead of bloating it.
